Detailed Description

The AppKit.TextKit.Formatter.FormatDescriptor defines a language formmating instruction that can be used by a AppKit.TextKit.Formatter.LanguageDescriptor to define syntax highlighting to a AppKit.TextKit.Formatter.LanguageFormatter.
