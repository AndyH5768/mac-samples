Returns
The list of word completions that will be presented to the user.
Parameters
textViewThe source AppKit.TextKit.Formatter.SourceTextView.
wordsA list of default words automatically provided by OS X in the user's language.
charRangeThe cursor location where the partial word exists.
indexThe word that should be selected when the list is displayed (usually 0 meaning the first item in the list). Pass -1 for no selected items.
