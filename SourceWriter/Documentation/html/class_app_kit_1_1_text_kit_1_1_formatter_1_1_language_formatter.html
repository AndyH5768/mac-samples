Detailed Description

The AppKit.TextKit.Formatter.LanguageFormatter class uses a AppKit.TextKit.Formatter.LanguageDescriptor to provide syntax highlighting of a NSTextField based on a set of AppKit.TextKit.Formatter.KeywordDescriptor and AppKit.TextKit.Formatter.FormatDescriptor objects..

Please see our Working with Text Controls Docs for more details: https://developer.xamarin.com/guides/mac/user-interface/standard-controls/#Working_with_Text_Controls
