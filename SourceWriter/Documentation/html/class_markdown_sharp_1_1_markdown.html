Detailed Description

Markdown is a text-to-HTML conversion tool for web writers. Markdown allows you to write using an easy-to-read, easy-to-write plain text format, then convert it to structurally valid XHTML (or HTML).

Create a new Markdown instance and optionally load options from a configuration file. There they should be stored in the appSettings section, available options are:

List<Token> MarkdownSharp.Markdown.TokenizeHTML ( string  text)
private

returns an array of HTML tokens comprising the input string. Each token is either a tag (possibly with nested, tags contained therein, such as <a href="&lt;MTFoo&gt;">, or a run of text between tags. Each element of the array is a two-element array; the first is either 'tag' or 'text'; the second is the actual value.
