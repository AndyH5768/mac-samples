Detailed Description

A custom segue type that replaces the existing View Controller with the new View Controller. This is different than most view controllers that push and pop the new View Controller on and off the stack.

This segue type is good for creating Preference Windows. See our Preference Window documentation for more details: https://developer.xamarin.com/guides/mac/user-interface/working-with-dialogs/#Creating_a_Preferences_Dialog
