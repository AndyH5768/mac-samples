Detailed Description

Manual toolbar item allows the developer to take control of the enabled/disabled state of a toolbar item.

See our Toolbar documentation for more infomation: http://developer.xamarin.com/guides/mac/user-interface/working-with-toolbars/#Disabling_Toolbar_Items
