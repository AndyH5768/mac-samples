Detailed Description

A custom NSSplitViewController that handles the T:MacInspector.ContentViewController displayed on the left side of the split and the Inspector (handled by a T:MacInspector.InspectorViewController) on the right side.

This class uses key-value coding and data binding to set the properties. Please see: https://developer.xamarin.com/guides/mac/application_fundamentals/databinding/
