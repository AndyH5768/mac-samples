Detailed Description

A custom segue type that replaces the existing View Controller with the new View Controller. This is different than most segues that push and pop the new View Controller on and off the stack.

This segue type is used to manage Inspector Panels. Please see: https://developer.xamarin.com/guides/mac/user-interface/working-with-windows/#Inspectors https://developer.xamarin.com/guides/mac/platform-features/storyboards/quickstart/
