Detailed Description

Controls the Inspector Panel displayed in the right hand side of the T:MacInspector.MainSplitViewController. This controller is responsible for switching out Inspector Panels as the context changes inside of the app.

Please see: https://developer.xamarin.com/guides/mac/user-interface/working-with-windows/#Inspectors https://developer.xamarin.com/guides/mac/platform-features/storyboards/quickstart/
